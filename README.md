# Rust Projects

This repo contains small projects and exercises made while learning the language rust.

## Projects

- **rpn/** — Reverse Polish Notation calculator (CLI)
- **game-of-life/** — Conway's Game of Life using macroquad

## How to run

Each project is a standalone Cargo project.

for example if you want to test rpn :
```bash
cd rpn
cargo run -- "3 4 +"
```
